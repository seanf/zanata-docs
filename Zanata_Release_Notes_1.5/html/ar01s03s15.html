3.15. Feature: Newline validation on leading and trailing string

Background:
Newlines in translation units do not require to be translated, but they should be preserved to preserve the formatting of the text.
New feature:
The Zanata 1.5 Translation Editor presents a validation error message when newlines in the translated unit do not match those in the source unit.
More information:
For more information refer to https://bugzilla.redhat.com/show_bug.cgi?id=768802.
