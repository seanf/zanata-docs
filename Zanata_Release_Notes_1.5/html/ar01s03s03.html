3.3. Feature: Bookmark a specific document

Background:
Prior to Zanata 1.5, it was possible to bookmark a workspace, but not a specific document.
New feature:
Documents are now presented in a way that allows a specific document to be bookmarked using a web browser bookmark.
More information:
For more information refer to https://bugzilla.redhat.com/show_bug.cgi?id=757621.
