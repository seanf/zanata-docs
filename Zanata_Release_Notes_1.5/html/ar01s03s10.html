3.10. Feature: Login to a particular project

Background:
Contributing translations requires a validated account on the Zanata server. Viewing a list of projects is possible without logging in. Prior to Zanata 1.5, when a user logs in to the Zanata server while viewing a project, they are taken to the Zanata server homepage.
New feature:
Logging in to a Zanata 1.5 server from a project page takes the user back to the project page after logging in.
More information:
For more information refer to https://bugzilla.redhat.com/show_bug.cgi?id=759001.
