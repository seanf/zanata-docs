5.3. Maintainers and Admins can add versions to non-active projects

Description:
Read-only and obsolete projects should be frozen, with no ability to add new versions. Currently Maintainers and Administrators can add versions to read-only projects though the web interface, and Administrators can add versions to obsolete projects through the web interface.
