5.4. mvn zanata:publican-push is rejected due to 401 (Unauthorized) when user is not the project maintainer

Description:
Only a project maintainer can push strings to a project. When a user other than a project maintainer attempts to push strings to a project using the maven plugin, a 401 error is produced. The wording of the error message does not make it clear that the user must be a project maintainer to successfully push the strings.
More Information:
