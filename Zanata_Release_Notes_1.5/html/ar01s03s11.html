3.11. Feature: Mark a project as obsolete

Background:
Translation projects cannot be deleted from a Zanata server. However, sometimes a project is no longer required, and the project maintainer may wish to make this explicit to translators.
New feature:
Projects can now be marked as obsolete, which will remove them from the list of projects presented to translators.
More information:
For more information refer to https://bugzilla.redhat.com/show_bug.cgi?id=750673.
