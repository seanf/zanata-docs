4.7. Bugfix: Long word in source cell invades the editor cell

Cause:
Translation cells in the translation editor did not have a scroll bar.
Consequence:
When a line in the source strings could not be wrapped it extended outside the translation cell and into the adjacent one.
Fix:
A horizontal scroll bar was added to the translation cell.
Result:
Long source strings that do not wrap are now presented with a scroll bar, and do not extend into the adjacent translation cell.
More information:
For more information refer to the original bug report at https://bugzilla.redhat.com/show_bug.cgi?id=759337#c4
